TCC - Aplicativo Educativo de Treino 
- Este projeto faz parte do Trabalho de Conclusão de Curso (Projeto Final) em Análise e Desenvolvimento de Sistemas. O objetivo é desenvolver um protótipo de aplicativo que auxilia usuários na prática de exercícios através de recomendações baseadas em similaridade biomecânica.

O Diferencial (IA de Recomendação)
- Diferente de apps comuns, este artefato utiliza filtragem baseada em conteúdo para sugerir trocas de exercícios em tempo real quando um equipamento está ocupado, garantindo que o estímulo muscular (plano de movimento e grupo alvo) seja preservado, além de ser um aplicativo mais voltado para o aprendizado da melhor execução possivel.

Estrutura do Repositório
- /app: Código-fonte desenvolvido em Flutter/Dart.
- /planning: Diagramas UML, modelagem de dados e documentos de planejamento.

Como rodar o projeto
1. Tenha o SDK do Flutter instalado.
2. Navegue até a pasta do app: `cd app`
3. Execute o comando: `flutter run`
